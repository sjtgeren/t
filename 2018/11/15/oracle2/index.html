Oracle
数据库：存放数据的仓库
why?
1)保存数据
file是轻量级保存数据。
数据库是重量级保存数据需要安装软件。
2)管理数据，有效增删改查(CRUD)，
使用SQL语句来操作。
3)在项目开发中，运行在项目的后台
。
项目分前台和后台。前台做界面。
如：前台注册界面
后台保存数据—保存到数据库中
大型数据库：oracle,db2 银行系统(
数据量大，安全性高，需要付费)
中型数据库：sql server,mysql 网站开发(
数据量大，安全性相对不高，免费的)
小型数据库: access,sqlite(
数据量小，安全性低，免费的)

CRUD。

数据库的发展：
1）层次模型的数据库:相当于树
2）网状模型的数据库:相当于蜘蛛网
3）关系型数据库：二元关系(二维表)
表与表之间的关系。

主键：primary key(PK)用来唯一标识表中的数据。
1)主键不能为空
2)主键的值必须唯一
外键：foreign key(FK)维护表与表之间的关系。
1)外键的值可以为空
2)外键的值不为空，
其值必须在所依赖的表中依赖的字段值存在。

一个认知：认知Oracle？
Oracle是关系型数据库。
Oracle是一家提供产品和服务的厂商

二个概念：
数据库？保存数据以及有效的管理数据
关系型数据库？对象与对象之间存在关系
。
使用二维关系保存对象。
三个名词：
sql:第四代语言，只关心做什么，
而不关心怎么做。
sqlplus:工具，执行sql或者pl/sql的命令并
显示执行语句后的结果
pl/sql:过程化查询语言
Buffer:存放SQLPLUS工具上最新执行的一条sql语句。
四种约束：
实体性完整性约束：针对主键，主键的值必须
非空唯一。
参照性完整性约束：针对外键，外键的值可以
为空，如果外键的值不为空，它
的值
必须在它依赖的表中依赖的字段值中存在
列级约束：它的值必须匹配它所定义的数据类型
用户自定义约束：check,如gender (男，女)
五种分类:针对sql命令
查询语句
select
数据操作语言:DML
insert,update,delete
数据定义语言：DDL
create,alter,drop,rename,truncate
事务控制语句
commit,rollback,savepoint
数据控制语言：DCL
grant(授权),revoke(回收权限)
六种对象：数据库中的对象
表：数据库最基本的结构
视图：一张虚表，从一张表或多张表查询的数据
组成的一张虚表。
序列：主要用来生成主键的值
索引：主要用来提供检索的效率
同义词：为对象取别名
存储单元：PL/SQL
。

查询部门表所有的信息？

字段名:直接根据字段名查找数据

2)数据库在算术上的使用
a)数字类型的算术运算：+，-，，/
查询所有员工的last_name和年薪？
select last_name,salary12
from s_emp;
查询所有员工的last_name和年薪(年终奖500)?
select last_name,salary12+500
from s_emp;
查询所有员工的last_name和年薪(月薪涨100)?
select last_name,(salary+100)12
from s_emp;
查询所有员工的last_name和年薪(包含提成)？
salary,提成：commission_pct
select last_name,salary,commission_pct,
(salary+commission_pct)12 sal
from s_emp;
b)日期类型的算术运算:+，-,单位是天
查询所有员工的入职日期和入职日期+1的信息？
select start_date,start_date+1
from s_emp;
3)取别名：为查询出来的字段重新取一个名字
a)在需要取别名的字段后空格，然后加上别名
select last_name,(salary+100)12 sal
from s_emp;
b)使用as关键字
select last_name,(salary+100)12 as Sal
from s_emp;
c)使用””,可以区分大小写和特殊字符
select last_name,(salary+100)12 “Sal”
from s_emp;
select last_name,(salary+100)12 “年薪”
from s_emp;
4)数据库处理空值
nvl(可能出现空值的字段,默认值)
如果可能出现空值的字段为空，
nvl函数处理的结果为默认值。
如果可能出现空值的字段不为空，
nvl函数处理的结果为本身的值。
select last_name,
(salary+nvl(commission_pct,0))12 sal
from s_emp;
5）字段拼接：||
查询last_name,first_name取别名name
select last_name||’,’||first_name name
from s_emp;
注意：字符串使用单引号
练习：
查询last_name-first_name in dept_id dept
select last_name||’-‘||first_name
||’ in ‘||dept_id||’ dept’
from s_emp;

h)sqlplus在文件的命令
save filename:把buffer中命令保存到文件中
如：save D:/test.sql
get filename:把文件中命令显示在工具上
如：get D:/test.sql
@ filename/start filename:执行脚本文件
如：start D:/test.sql
@ D:/test.sql
edit filename:将文件以编辑的方式打开
如：edit D:/test.sql
spool filename:将工具中所有内容都写入到文件中
如： spool D:/test1.sql
select last_name from s_emp;
show user
select * from s_dept;
spool off;
注意：spool必须和spool off一起使用。
exit:退出
