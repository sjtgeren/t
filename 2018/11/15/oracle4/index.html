转换函数
to_char():将数字类型或者日期类型转换为字符串类型
1)将日期类型转换为字符串类型
select to_char(sysdate,’yyyy’)
from dual;
select to_char(sysdate,’ddspth’)
from dual;
2013-08-19 16:16:30 pm
select to_char(sysdate,
‘yyyy-MM-DD HH24:MI:SS pm’)
from dual;
2）将数字类型转换为字符串类型
select to_char(salary,’fmL000.00’)
from s_emp;
9：位数不满足定义的位数时，不需要强制补齐
0：位数不满足定义的位数时，需要强制补齐
$:西方的货币符号
L:本地的货币符号
,:分割
.:小数点
fm:去掉前面空格
注意：定义的位数小于实际的长度会显示#
to_number()：将字符串类型转换为数字类型
select to_number(‘5’) from dual;
to_date()：将字符串类型转换为日期类型
1)如果日期是默认格式
select to_date(‘19-AUG-13’)
from dual;
2)如果日期不是默认格式，就需要
对格式进行说明。
select to_date(‘2013/08/19’,
‘yyyy/MM/DD’)
from dual;

month: 英文月份(例如:march(3月))

DY: 星期几的前三个字母(例如:mon)

ddspth:英文拼写的第几天(例如:fifth(第五天))
d: 一个星期的数字表示，星期天为1，星期六为7
dd: 一个月的第几天
ddd: 一年的第几天
yyyy:四位数的年
mm:二位数的月
hh24:24小时制
mi:分钟
ss:秒

实质：是一张表的查询

连接方式：
1）等连接：连接条件使用=连接
例一：查询所有员工的ID，名字和所在部门的名称
步骤一：分析查询的表
s_emp s_dept
步骤二：连接条件
s_emp.dept_id=s_dept.id
步骤三：写sql语句
select e.id,e.last_name,d.name
from s_emp e,s_dept d
where e.dept_id=d.id;

例二：查询员工的姓名和部门所在地区的名称(3张表)
步骤一：查询的表
s_emp s_dept s_region
步骤二：连接条件
s_emp.dept_id=s_dept.id
s_dept.region_id=s_region.id
步骤三：拼接sql语句
select e.last_name,d.name,r.name
from s_emp e,s_dept d,s_region r
where
e.dept_id=d.id and
d.region_id=r.id;

第五章
avg:求平均值
count:求个数，非空个数
max:求最大值
min:求最小值
sum:求和
stddev:求标准差
variance:求方差

例一：查询所有员工的平均工资，最高工资最低工资，工资总和，还有有多少个员工?
select avg(salary),max(salary),min(salary),sum(salary),
count() from s_emp;

例二：查询每个部门的平均工资？对平均工资降序排序.平均工资大于1400.
简化1：求41号部门的平均工资？
select avg(salary) from s_emp
where dept_id=41;
简化2：求每个部门的平均工资？
group by–>值相同的数据放在一组
select dept_id, avg(salary) from s_emp
group by dept_id;
简化3：平均工资大于1400？
select dept_id, avg(salary) from s_emp
where avg(salary)>1400
group by dept_id;//error
select dept_id, avg(salary) from s_emp
group by dept_id
having avg(salary)>1400;

简化4：平均工资按照降序排列？
select last_name,dept_id, avg(salary) a from s_emp
group by dept_id,last_name
having avg(salary)>1400
order by a desc;
注意：
1.group by进行分组，值相同的数据放在一组，放在where之后
2.在where中不能够出现组函数，通过having进行过滤
放在group by后面
3.出现在select后面的字段，如果该字段不是组函数，就必须在group by后面声明
4.多个字段分组，多个字段值必须都相同
